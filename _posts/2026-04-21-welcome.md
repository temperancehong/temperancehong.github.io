This is the first post on the rebuilt site. Writing here is deliberately easy:
I drop a markdown file into `_posts/`, commit, and GitHub Pages rebuilds.

## How posts work

Each file in `_posts/` is named `YYYY-MM-DD-some-title.md` and starts with
frontmatter:

```yaml
---
title: "My post title"
date: 2026-05-03
---
```

Below the frontmatter, anything in markdown renders — headings, **bold**,
*italic*, `code`, [links](https://example.com), code blocks, lists, quotes.

> This is a blockquote, for example.

## Why a new site?

The old page was a half-finished student project. This one is a clean slate
